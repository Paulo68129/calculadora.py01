# Calculadora em Python com Programação Orientada a Objetos (POO)

Este projeto consiste no desenvolvimento de uma **calculadora simples em Python**, utilizando os conceitos de **Programação Orientada a Objetos (POO)**.  
A aplicação realiza as **operações matemáticas básicas** e está disponível em **duas versões**: via **terminal** e via **interface gráfica**.

O objetivo principal é demonstrar, de forma prática, a aplicação de classes, métodos e encapsulamento em Python.

---

## 📌 Funcionalidades

- ➕ Soma  
- ➖ Subtração  
- ✖️ Multiplicação  
- ➗ Divisão  
- Tratamento de erros:
  - Entrada inválida de dados
  - Divisão por zero
- Duas formas de execução:
  - Versão em **terminal**
  - Versão com **interface gráfica (Tkinter)**
